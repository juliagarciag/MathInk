# ✨ Sobre o projeto

O MathInk é uma calculadora moderna desenvolvida para unir:

- inteligência artificial
- design futurista
- matemática
- reconhecimento de escrita à mão

O projeto possui diferentes modos de cálculo e uma interface premium inspirada em aplicações modernas.



# 🚀 Funcionalidades

✅ Calculadora Tradicional  
✅ Calculadora Científica  
✅ Ink Mode (desenho de contas à mão)  
✅ Reconhecimento de escrita com IA  
✅ Correção manual da expressão detectada  
✅ Explicação passo a passo  
✅ Interface futurista responsiva  
✅ Glassmorphism UI  



# 🤖 Inteligência Artificial

O projeto utiliza a API Gemini para:

- reconhecer contas desenhadas à mão
- interpretar expressões matemáticas
- gerar explicações simples passo a passo



# 🎨 Interface

O visual do projeto utiliza:

- glassmorphism
- gradientes futuristas
- animações suaves
- efeitos interativos modernos



# 🛠️ Tecnologias utilizadas

- HTML5

- FIGMA
